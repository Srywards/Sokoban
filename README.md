make && ./my_sokoban maps/map

The game has to be played with the arrow keys (LEFT, RIGHT, UP and DOWN).

You can create your own map but a valid map can only contains the characters SPACE, ‘\n’, ‘#’, ‘X’, ‘O’ and ‘P’.
